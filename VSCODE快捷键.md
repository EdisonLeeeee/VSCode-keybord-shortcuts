## 通用快捷键(General)
|快捷键|描述|
|-|-|
|Ctrl + Shift + P, F1|显示命令面板
|Ctrl + P|快速打开/转到文件
|Ctrl + Shift + N|打开一个新窗口/实例
|Ctrl + Shift + W|关闭一个新窗口/实例
|Ctrl + ,|打开用户设置
|Ctrl + K Ctrl + S|键盘快捷方式
## 基本编辑快捷键(Basic editing)
|快捷键|描述|
|-|-|
|Ctrl + X|剪切
|Ctrl + C|复制
|Alt +  ↑ / ↓|向上/向下移动该行
|Shift + Alt  +  ↓ / ↑|向下/向上复制该行
|Ctrl + Shift + K|删除该行
|Ctrl + Enter|向下插入一行
|Ctrl + Shift + Enter|向上插入一行
|Ctrl + Shift + \\|跳转到匹配的括号前方
|Ctrl +  ] / [|整行缩进/凸进
|Home / End|跳转至行首/行尾
|Ctrl + Home|跳转到文件开头
|Ctrl + End|跳转到文件结尾
|Ctrl + ↑ / ↓|滚动条向上/向下滚动
|Alt + PgUp / PgDn|滚动条向上/向下翻页滚动
|Ctrl + Shift + [|折叠该区块
|Ctrl + Shift + ]|展开该区块
|Ctrl + K Ctrl + [|折叠所有子区块
|Ctrl + K Ctrl + ]|展开所有子区块
|Ctrl + K Ctrl + 0|折叠所有区块
|Ctrl + K Ctrl + J|展开所有区块
|Ctrl + K Ctrl + C|注释
|Ctrl + K Ctrl + U|取消注释
|Ctrl + /|切换行注释
|Shift + Alt + A|切换块注释
|Alt + Z|切换换行
## 导航(Navigation)
|快捷键|描述|
|-|-|
|Ctrl + T|显示所有符号
|Ctrl + G|跳转到行
|Ctrl + P|跳转到文件
|Ctrl + Shift + O|跳转到符号
|Ctrl + Shift + M|显示问题面板
|F8|跳转到下一个错误或警告
|Shift + F8|跳转到前一个错误或警告

|Alt +  ← / →|向后/向前浏览标签页
|Ctrl + M|切换Tab移动光标
## 搜索替换(Search and replace)
|快捷键|描述|
|-|-|
|Ctrl + F|查找
|Ctrl + H|替换
|F3 / Shift + F3|查找下一个/上一个
|Alt + Enter|选中所有查找到的匹配
|Ctrl + D|增加选中新的匹配
|Ctrl + K Ctrl + D|移动至下一个新的匹配
